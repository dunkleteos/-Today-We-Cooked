# 🍔 Today We Cooked

## 👥 Team
- **Deniz Murat** – Game design, Coding  
- **Efe Özgür** – Coding, Level Design, Music
- **Ömür Çamurcu** – , UI Art, Animations, Sprites, Art design 

**Today We Cooked** is a 2D platformer / mini-game inspired by a game jam project where the player, a chef, collects ingredients to make the ultimate burger.

---

## 🎮 Gameplay
- Climb towers and explore levels in a time-limited platformer stage.
- Fight inside a top-down pixel arena to earn your ingredients.
- Try to satisfy a picky food critic and find the perfect sauce recipie.

---

## 🛠 Tech Stack
- Unity 6
- C#
- Unity Input System
- Animator State Machine
- 2D Physics
